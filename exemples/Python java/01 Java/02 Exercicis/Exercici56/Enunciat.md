### Exerici 56

Fes una funció (porcio_taula) que rebi tres paràmetres:

* Número
* Valor inici
* Valor final

La funció ha d'escriure la porció de la taula de multiplicar entre 'inici' i 'final'

Exemples:

```python
porcio_taula(2, 4, 7)
4 x 2 = 8
5 x 2 = 10
6 x 2 = 12
7 x 2 = 14

porcio_taula(5, 3, 5)
3 x 5 = 15
4 x 5 = 20
5 x 5 = 25
```
